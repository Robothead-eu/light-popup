# Light Popup

Lightweight, privacy-first WordPress popup plugin. Free, open-source, distributed via GitHub.

Built on the [Robothead WordPress Plugin Template](https://github.com/robothead/robothead-wordpress-plugin-template-standard).

## Requirements

- PHP 8.1+
- WordPress 6.0+
- Composer

## Setup

```bash
composer install
```

## Development

Follows the Robothead Plugin Standard — Composer-based, PSR-4 namespaced, thin-bootstrap structure.
